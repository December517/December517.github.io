HTML骨架

标签含义

  • html : 最根本的标签,表示整个的网页.
  • head : 网页的头部, 里面都是一些网页的配置, - 除了 title 标签,其他的内容在浏览器上都是 不可见的.
  • title : 网页的标题. (网页的名称)
  • body : 用户能够看到的内容区域, 网站的内容.

html 标签是一个双标签

  • 双标签 : 有开始签和结束标签. 必须是成双出现的.
  • html 标签有两个属性.
  • xmlns , 是 xml 的namespace 规定了我们的标签使用的是那种命名规范. 用于浏览器不一致, 必须规范死什么是标签什么是内容,标签具有什么语义.
  • lang : 规定了html文件标签都是英文的.

head标签

·head 标签也是一对双标签, 里面是对我们网页进行的一个配置.

meta 标签 : 单标签

  • 红色 meta 标签规范是我们网页使用的字符集.

title 标签 : 双标签 , 内部书写的是网页的标题.

搜索引擎优化 : seo (提升我们的搜索排名 )
title 里的文字会作为关键字首先被搜索引擎抓取.

body标签 : 双标签

body 元素定义文档的主体.
body 元素包含文档的所有内容（比如文本、超链接、图像、表格和列表等等.）

HTML 的语法

  • 在编辑 html 的时候, 标签对换行和空格不敏感,只对标签之间的嵌套敏感. 为了保证我们代码的可读性, 需要对标签进行合理的缩进.
  • 空白折叠现象, 在普通文本内部 :如有空格. 缩进. 换行 , 将这些空白区域折叠成一个空格显示在浏览器里.

标签分类

容器级 : 内部还可以放置嵌套任意类型的标签,甚至是容器级的标签. div , h 系列.
文本级 : 只能放文字, 图片, 表单元素, 废弃文字修饰标签等. 不能放容器级标签. p 标签.

标签属性

  • 每一个标签都有自己特定的属性.
  • 属性都有属性值, 属性名k(key), 属性值v(value), 键值对写法
  1. 属性书写在标签的起始标签内部, 与标签之间用空格隔开属性与属性之间也需要用空格隔开.
  2. 键值对书写格式 : k=”v”. 中间不能有空格.
  3. 属性值必须是用双引导号包裹, xhtml 严格要求使用双引导号.
  4. 标签属性值可以有一个或者多个,多个属性值之间用空格隔开,全部必须在引号内部.
